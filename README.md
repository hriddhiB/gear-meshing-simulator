# Gear Meshing Simulator

An interactive simulator that visualizes gear meshing mechanics and rotational motion.

https://your-app-name.streamlit.app

## Features
- Interactive gear rotation
- Adjustable number of teeth

- Real-time visualization

## Technologies Used
- Python
- Streamlit
- Plotly
- Mechanical engineering concepts

## Educational Purpose
Helps students understand gear meshing, gear ratio, and rotational motion in mechanical systems.
